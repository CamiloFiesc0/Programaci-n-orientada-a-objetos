Descripción del Proyecto

Librería CUN es un software de gestión desarrollado en Python que permite administrar productos, categorías y existencias dentro de una librería o papelería.
El sistema facilita el registro, consulta, modificación y eliminación de productos, además de generar reportes y respaldos automáticos.
Fue diseñado como un proyecto académico para aplicar los principios de análisis, diseño y desarrollo de software.
